# XHS Video Note Agent  
**版本**：0.1.5  

> **视频转为优质小红书爆款笔记 AI 工具**  
> Upload a Xiaohongshu (小红书) video link or text, automatically fetch transcript, and generate a platform-style viral note.

---

## 一、功能概览

- **支持小红书链接输入**  
  用户在「开始」输入框粘贴小红书分享链接或视频文字信息（text），可选指定输出语言（language）。

- **自动清洗与转录**  
  1. 去除特殊字符（“输入文本特殊字符处理”节点）。  
  2. 根据链接调用 `{{base_url}}/get_xhs_video_transcript` 接口，获取视频字幕文本。

- **AI 文案生成**  
  - **小红书爆款文案**  
    使用 GPT-4o 专业小红书文案模板，将转录文字重组为「标题 + 正文 + 标签」格式的爆款笔记。  
  - **图生图提示**  
    将最终小红书文案提炼为「主体／动作／背景／色彩」四部分英文描述，准备图像生成。

- **Flux 图像生成**  
  最后调用内置 Flux（模型：schnell，分辨率：768×1024）根据英文提示生成配图。

- **结果输出**  
  - `text`：生成的小红书笔记完整文案。  
  - `files`：一张 AI 生成的配图（JSON 数组）。

---

## 二、输入参数

| 参数       | 类型     | 是否必填 | 说明                                       |
|------------|----------|----------|--------------------------------------------|
| `text`     | `string` | 是       | 小红书分享链接或视频文字内容（可包含短链接） |
| `language` | `string` | 否       | 输出文案语言，例如 `zh` / `en` 等           |

> **环境变量**  
> - `base_url`：视频转录服务地址，默认 `https://iagent.cc/video-analysis-api`

---

## 三、工作流步骤

1. **开始（Start）**  
   接收 `text` 和可选 `language`。

2. **清洗文本**  
   执行 Python 代码节点去除换行等特殊字符。

3. **调用接口转录**  
   根据清洗后的 `text` 调用小红书转录接口，获取 `body` 文本。

4. **生成小红书文案**  
   GPT-4o 节点：将 `body` 转换为爆款笔记风格文案。

5. **提炼图生图提示**  
   GPT-4o 节点：将中文文案拆分为主体、动作、背景、色彩四部分英文描述。

6. **Flux 图像生成**  
   使用 Flux 根据英文描述生成一张配图。

7. **结束（End）**  
   输出 `text`（小红书笔记文案）和 `files`（生成的图片文件数组）。

---

## 四、快速使用

1. **部署到 Dify 平台**  
   - 在 Dify 控制台 “创建应用” → 上传 `XHS Video Note Agent.yml`  
   - 配置环境变量 `base_url`（如有私有部署需求）

2. **调用流程**  
   - 填写 `text`（如 `http://xhslink.com/…`）  
   - 可选填写 `language`（如 `en`）  
   - 点击 Start → 获取生成文案与配图
